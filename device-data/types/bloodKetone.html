data-model

Blood Ketones (bloodKetone)

NB: All fields are required unless otherwise noted.

type

[ingestion, storage, client] The string bloodKetone.

Blood ketones represent ketone concentration values (specifically beta-ketones, primarily beta-hydroxy butyric acid) obtained from a fingerstick meter capable of reading specialized blood ketone testing strips. Tidepool uses a separate data model for urine ketones.

units

[ingestion, storage, client] The string mmol/L.

value

[ingestion, storage, client] Blood ketone value in mmol/L (float, potentially unrounded), with appropriately matching units field.
