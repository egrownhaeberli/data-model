Urine Ketones (urineKetone)

NB: All fields are required unless otherwise noted.

type

[ingestion, storage, client] The string urineKetone.

Urine ketones represent ketone concentration values obtained by home testing with ketone reagent strips, usually reacting positively to quantities of acetoacetic acid (rather than beta-ketones) in the urine. These records differ significantly from blood ketone records because the values obtained are qualitative, not quantitative.

value

[ingestion, storage, client] String value representating a qualitative measurement of urine ketone concentration.

Must be one of: negative, trace, small, moderate, large.
